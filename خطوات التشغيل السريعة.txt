### الخطوة 1: تثبيت المكتبات المطلوبة
افتح Command Prompt أو Terminal واكتب:

```bash
pip install streamlit plotly openpyxl pandas
```

في نفس المجلد اللي فيه الملفات، اكتب:

```bash
python -m streamlit run dashboard_FIXED_FINAL.py

```
